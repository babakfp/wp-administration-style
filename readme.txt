این افزونه, فونت همه‌ی محتوای متنی پنل ادمین وردپرس را به فونت فارسی وزیر تغییر میدهد و همچنین حالت راست‌چین بودن و ظاهر پنل ادمین را برای زبان فارسی بهبود میبخشد.




== Description ==
فونت فارسی وزیر را به پنل ادمین وردپرس اضافه میکند تا برای زیبایی پنل ادمین, محتوای متنی پنل ادمین با فونت وزیر نمایش داده شود. همچنین حالت راست‌چین بودن و ظاهر پنل ادمین را برای زبان فارسی بهبود میبخشد.
* با گوتنبرگ سازگار است
